## 🧩 Common Windows Problems

### The file will not open
- Check that the download finished
- Try opening the file again
- If it is a `.zip`, extract it first
- Right-click the app and choose Run as administrator

### Windows blocks the app
- Open the file again
- Choose More info if you see it
- Select Run anyway if you trust the release page

### The app opens but shows no files
- Check that you picked the right folder
- Make sure the folder contains the files you want to manage
- Refresh the library or restart the app

### Search does not find an item
- Check the title and tags
- Use fewer words
- Try the file name instead of a full sentence
